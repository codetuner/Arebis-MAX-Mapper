2. Files to be installed in the GAC
-----------------------------------

Max.Tools.DomainGenerator.15.0.dll

3. Shortcuts to create
----------------------

Create a shortcut "Start Menu/Programs Menu/Arebis/MAX Extensions for VS.NET 2017/Max Mapping Generation Manual"
pointing to the manual file.

4. Registry settings
--------------------

On 32-bit machine:

   Windows Registry Editor Version 5.00
   
   [HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\.NETFramework\v4.0.30319\AssemblyFoldersEx\Arebis MAX Mapper Extensions]
   @="C:\\Program Files (x86)\\Arebis\\MAX Extensions for VS.NET 2017\\Assemblies\\"

On 64-bit machine:

   Windows Registry Editor Version 5.00
   
   [HKEY_LOCAL_MACHINE\SOFTWARE\WOW6432Node\Microsoft\.NETFramework\v4.0.30319\AssemblyFoldersEx\Arebis MAX Mapper Extensions]
   @="C:\\Program Files (x86)\\Arebis\\MAX Extensions for VS.NET 2017\\Assemblies\\"
